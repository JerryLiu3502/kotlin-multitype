# 贡献指南

欢迎为 Kotlin MultiType 贡献代码！

## 如何贡献

1. **Fork** 本仓库
2. 创建你的特性分支 (`git checkout -b feature/amazing-feature`)
3. 提交你的更改 (`git commit -m 'Add some amazing feature'`)
4. 推送分支 (`git push origin feature/amazing-feature`)
5. 创建 **Pull Request**

## 代码规范

- 使用 Kotlin 官方代码风格
- 确保所有测试通过
- 添加新功能时请同时添加单元测试
- 更新文档（如果有需要）

## 开发环境

- Kotlin 1.9.x
- Gradle 8.5+
- JDK 17+

## 运行测试

```bash
./gradlew test
```

## 构建项目

```bash
./gradlew build
```

## 发布

```bash
./gradlew publish
```
